##  Installation

Följ stegen nedan för att köra projektet lokalt:

```bash
# Klona projektet
https://github.com/sanhu7/react-app-fixed

# Gå in i mappen
cd react-app-fixed

# Installera beroenden
npm install

# Starta utvecklingsserver
npm run dev
